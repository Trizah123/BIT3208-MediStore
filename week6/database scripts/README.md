# BIT3208 Week 6 - Medical Store CRUD Operations

## Project Description
A PHP and MySQL web application for managing medicine inventory.
Implements full CRUD operations (Create, Read, Update, Delete).

## Student Details
- Course: BIT3208 - Advanced Web Design and Development
- Week: 6 — Database Integration and CRUD Operations
- Project: Medical Store Management System

## Features
- Add new medicines
- View all medicines in a table
- Edit/update medicine records
- Delete medicines
- Low stock warning (quantity below 20 shown in red)

## Files
- source_code/ — All PHP files
- database_scripts/ — SQL file to set up the database
- screenshots/ — Screenshots of the working system

## How to Run
1. Install XAMPP and start Apache and MySQL
2. Open phpMyAdmin and import database_scripts/medical_store.sql
3. Copy source_code/ files to C:/xampp/htdocs/BIT3208-MediStore/week6/
4. Open browser and go to:
   http://localhost/BIT3208-MediStore/week6/view_medicines.php
