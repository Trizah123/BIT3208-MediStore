## Project Description
MediStore is a medicine inventory management system built from Week 1 to Week 5.

## Technologies Used
- HTML, CSS, JavaScript
- PHP
- MySQL
- XAMPP

## Features
- Login and logout system
- Database connection
- CRUD operations (Add, Edit, Delete medicines)

## How to Run
1. Copy to C:\xampp\htdocs\BIT3208-MediStore
2. Start XAMPP (Apache & MySQL)
3. Import database medistore_db
4. Go to http://localhost/BIT3208-MediStore/Week5/login.html
5. Login: admin / 1234
